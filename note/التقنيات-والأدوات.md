## أوامر المشروع

### أوامر التطوير
- `npm run dev`: تشغيل خادم التطوير على المنفذ 3000
- `npm run build`: بناء المشروع للإنتاج
- `npm run build:static`: بناء نسخة ثابتة (static export)
- `npm run start`: تشغيل النسخة المبنية

### أوامر الجودة
- `npm run lint`: فحص الكود بـ ESLint
- `npm run typecheck`: فحص أنواع TypeScript

### أوامر مخصصة
- `npm run app:init`: تهيئة التطبيق وتوليد الملفات العامة
- `npm run server:stop`: إيقاف خادم التطوير

## هيكل المشروع

```
gova/
├── src/
│   ├── app/           # صفحات Next.js
│   ├── components/    # المكونات القابلة لإعادة الاستخدام
│   ├── data/          # البيانات الثابتة
│   └── dev/           # أدوات التطوير
├── public/            # الملفات العامة
├── scripts/           # السكريبتات المساعدة
└── doc/               # التوثيق
```
